# Testing and Validation

## Objective
The purpose of testing was to verify that the vehicle met competition requirements while maintaining reliability, stability, safety, and repeatable performance.

## Validation Categories

### Performance Testing
The vehicle was tested repeatedly on the competition course. Testing focused on:

- Precision speed run performance
- Turning reliability
- Wheel traction
- Motor performance
- Launcher timing

### Safety Testing
The vehicle successfully completed:

- Static tipping tests in multiple directions
- Upside-down retention testing
- Teddy bear positioning verification
- Continuous line-of-sight requirement testing

### Mass Validation
The final measured mass was compared against the calculated design mass. Results showed a difference of approximately 35 g between prediction and final assembly.

## Final Results

| Metric | Result |
|----------|----------|
| Speed Run Time | 5.81 seconds |
| Speed Run Score | 30/30 |
| Eco-Trash Launch Score | 26/30 |
| Measured Mass | 1148 g |
| Calculated Mass | 1113 g |
| Static Tipping Test | Pass |
| Upside Down Test | Pass |
| Line of Sight Test | Pass |

## Lessons Learned
Repeated testing was critical for improving Arduino timing, launcher consistency, drivetrain reliability, and overall competition performance. The team identified several opportunities for weight reduction and improved manufacturing efficiency during testing.
