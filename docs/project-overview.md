# Project Overview

## Project Title
Assistive Mobility Device Engineering Project

## Original School Project Name
Teddy Bear Wheelchair Project

## Project Summary
This project involved designing and building a small-scale assistive mobility device prototype, commonly referred to during the school project as the Teddy Bear Wheelchair Project. The goal was to apply engineering design principles to create a functional wheelchair-style mobility device that could support and move a teddy bear as a model user.

## Purpose
The purpose of this project was to demonstrate the full engineering design process, including problem definition, brainstorming, prototyping, component selection, assembly, testing, and final documentation.

## Portfolio Purpose
The original school deliverables are preserved in the repository as untouched project evidence. The files in this documentation section organize and extract the key engineering information into a more professional format for portfolio, GitHub, and resume use.

## Core Engineering Areas
- Mechanical design
- Electrical integration
- Assistive mobility prototyping
- Arduino/control logic
- Testing and validation
- Budgeting and bill of materials documentation
- Team-based engineering design

## Final Outcome
The final project resulted in a physical wheelchair-style prototype intended to demonstrate assistive mobility concepts at a scaled-down level. The project showed how mechanical structure, movement systems, electronic control, and safety considerations can be combined into one functional engineering prototype.
