# Launcher System

## Overview
The launcher subsystem was designed for the Eco-Trash Launch event. The final design used a mousetrap-based launcher with a servo-controlled release mechanism.

## Main Components

- Mousetrap launcher core
- Servo-controlled trigger
- Launcher arm
- Projectile holder
- Side-mounted launch opening
- Supporting brackets and frame members

## Design Rationale
The mousetrap launcher was selected because it was simple, lightweight, inexpensive, and easy to integrate with the existing Arduino-controlled system.

## Integration with Vehicle
The launcher was positioned so that it extended from the side of the ambulance body. This kept the launcher clear of the teddy bear passenger, wiring, and major electronics.

## Performance
The launcher successfully completed the Eco-Trash Launch event and achieved a final score of 26/30.

## Challenges

- Launcher arm bending under added rubber band tension
- Inconsistent projectile direction and distance
- Servo interference with the hook/release mechanism
- Projectile shaking during movement over the course

## Improvements Made

- Projectile was changed from a bottle to a bottle-cap projectile
- Servo was repositioned lower to reduce misfires
- Launcher path was kept clear of the teddy bear and electronics

## Future Improvements

- Use a more rigid launcher arm
- Improve trigger clearance
- Reinforce the mousetrap mounting structure
- Create a more secure projectile holder
- Test projectile options earlier in the design process
- Add protective guarding around moving launcher components
