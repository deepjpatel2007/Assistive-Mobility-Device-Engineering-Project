# System Architecture

## Overview
The Assistive Mobility Device was designed as a multidisciplinary engineering system consisting of mechanical, electrical, software, and launcher subsystems.

## Mechanical Subsystem

Components:
- Three-wheel chassis
- Front drive axle
- Pulley drive system
- Wheel supports and frame members
- Teddy bear support structure

Responsibilities:
- Structural support
- Stability
- Power transmission
- Mobility

## Electrical Subsystem

Components:
- Arduino Uno
- L298N Motor Controller
- DC Motor
- Battery Pack
- Breadboard
- Wiring and passive components

Responsibilities:
- Power distribution
- Motor control
- Trigger activation
- Timing operations

## Software Subsystem

Components:
- Arduino control program
- Movement timing logic
- Launcher activation logic

Responsibilities:
- Vehicle movement control
- Competition timing sequence
- Servo activation

## Launcher Subsystem

Components:
- Mouse trap mechanism
- Servo trigger
- Launch platform

Responsibilities:
- Eco-trash launch event
- Reliable projectile release

## System Integration
The final design integrated all subsystems into a single competition vehicle optimized for speed, reliability, stability, and repeatable operation.
